## Integrantes

- Franco Ariel Martín

Proyecto React + Vite para la materia CIU.

## Requisitos

- Node.js
- npm

## Instalación

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

Luego abrir en el navegador la URL que indique Vite, por ejemplo:

```bash
http://localhost:5173/
```

## API Utilizada

Este proyecto utiliza la API provista por la consigna (Caja Negra).
