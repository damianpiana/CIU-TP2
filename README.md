# CIU TP2

Proyecto React + Vite para la materia CIU.

## Requisitos

- Node.js
- npm

## Instalación

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

Luego abrir en el navegador la URL que indique Vite, por ejemplo:

```bash
http://localhost:5173/
```

## Build de producción

```bash
npm run build
```

## Notas

- La aplicación usa React, React Router y React Bootstrap.
- `index.html` debe estar en la raíz del proyecto para que Vite lo encuentre.
- `dist/` es la carpeta de salida de la build y no debe versionarse.
