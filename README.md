# CIU TP2

Proyecto React + Vite para la materia CIU.

## Requisitos

- Node.js
- npm

## Instalación

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

Luego abrir en el navegador la URL que indique Vite, por ejemplo:

```bash
http://localhost:5173/
```

## Notas

- La aplicación usa React, React Router y React Bootstrap.
